# Simple HTML & CSS Website

## 🚀 Features

- Clean and semantic HTML structure
- Basic CSS styling
- Responsive layout
- Simple navigation between pages

## 📁 Pages

- `index.html` – Home page with introductory content
- `contact.html` – Contact page with a basic form or contact details
